# ✨ Hybee — Premium Fashion E-Commerce Platform

Hybee is a modern, full-stack fashion e-commerce platform designed with a focus on elegance, performance, and user experience. It delivers a seamless shopping journey with refined UI, structured product discovery, and an interactive styling feature.

---

## 🖤 About the Project

Hybee combines clean design with scalable architecture to create a premium digital shopping experience. From browsing curated collections to customizing outfits visually, the platform is built to reflect modern fashion standards.

---

## 🚀 Key Features

### 🛍️ Shopping Experience
- Browse products by **category, gender, and price**
- Detailed product pages with **ratings, variants, and gallery**
- Smooth **add-to-cart and checkout flow**
- Responsive design across all devices

### 👤 User System
- Secure authentication
- User profile and order tracking
- Persistent cart and session handling

### 🎨 Style Mirror (Signature Feature)
An interactive styling tool where users can:
- Upload a full-body image or use camera
- Drag and drop clothing and accessories
- Experiment with different outfit combinations
- Visualize their style before purchasing
